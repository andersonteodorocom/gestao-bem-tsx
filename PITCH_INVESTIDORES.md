# 🎯 Gestão do Bem - Pitch para Investidores

## 🔍 O Problema

O **terceiro setor** movimenta bilhões de reais anualmente e impacta milhões de vidas, mas enfrenta desafios críticos:

- **90%** das ONGs brasileiras operam com gestão manual e planilhas desconectadas
- Perda de **30-40%** do tempo em processos administrativos que poderiam ser automatizados
- Dificuldade em **mensurar e comprovar impacto social** para captação de recursos
- Alta rotatividade de voluntários por **falta de engajamento** e **organização**

## 💡 Nossa Solução

**Gestão do Bem** é uma plataforma SaaS completa para gestão de ONGs e organizações do terceiro setor, oferecendo:

### ✨ Diferenciais Competitivos

1. **Gestão Unificada** - Dashboard completo para voluntários, eventos, doações e tarefas
2. **Simplicidade** - Interface intuitiva, desenvolvida especificamente para o terceiro setor
3. **Rastreabilidade** - Logs de auditoria e histórico completo de atividades
4. **Escalável** - Arquitetura moderna que atende de pequenas ONGs a grandes organizações

### 🎯 Funcionalidades Core (MVP)

- ✅ Gestão de voluntários (cadastro, habilidades, disponibilidade)
- ✅ Organização de eventos e campanhas
- ✅ Sistema de tarefas com priorização
- ✅ Dashboard analítico em tempo real
- ✅ Controle de acesso e autenticação segura
- ✅ Recuperação de senha por e-mail

## 🚀 Tecnologia

### Stack Moderno & Escalável

**Backend**
- NestJS (Node.js) - framework enterprise-grade
- TypeORM - ORM robusto com suporte a migrations
- JWT Authentication - segurança corporativa

**Frontend**
- React + Vite - performance otimizada
- Shadcn/UI - componentes acessíveis e modernos
- TanStack Query - gerenciamento de estado eficiente

**Infraestrutura**
- MySQL - banco relacional confiável
- Docker - containerização e deploy simplificado
- RESTful API - integração facilitada
